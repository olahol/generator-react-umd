# generator-react-umd

> A minimalistic scaffold for a ES6 React component wrapped with the UMD.

Includes:

* [x] Compilation from ES6.
* [x] UMD wrapping.
* [x] Tests using `jsdom` and Mocha.
* [x] Dev server using BEEFY.

# Usage

Start the BEEFY dev server.

```bash
npm start
```

Run the Mocha tests.

```bash
npm test
```
